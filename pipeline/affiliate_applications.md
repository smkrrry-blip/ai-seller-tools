## 登録後の作業（Claudeが実行）

登録完了後、実際のアフィリエイトURLを以下のファイルに追記してください：
```
~/affiliate-en/pipeline/.env
```

フォーマット：
```
HELIUM10_LINK=https://imp.i368448.net/xxxxx
JUNGLE_SCOUT_LINK=https://junglescout.com/?partner=xxxxx
AMAZON_JP_TAG=xxxxx-22
AMAZON_US_TAG=xxxxx-20
```

記入後、ターミナルで：
```bash
python3 ~/affiliate-en/pipeline/replace_links.py
```
→ 全記事のリンクが一括で実URLに置き換わります。
