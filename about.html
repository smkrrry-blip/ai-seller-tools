About This Site

Hi, I'm Shoichi

I'm an Amazon FBA seller based in Japan. I've been selling on Amazon since 2019 — first on Amazon Japan, then expanding to the US and European marketplaces.

Why I Built This Site

When I started researching AI tools for my Amazon business, I ran into the same problem over and over: most reviews were written by people who had never actually sold anything on Amazon. They'd list features, copy marketing copy, and slap on affiliate links.

I wanted a resource written by someone who actually uses these tools in a real business, every day. So I built it.

My Experience

  • Amazon FBA seller since 2019 (5+ years)
  • Active on Amazon JP, US, and EU marketplaces
  • I personally pay for and test every tool I review
  • I cover both the strengths and the weaknesses — including tools I've cancelled

Editorial Standards

Every review on this site is based on my personal experience. I note how long I've used each tool, what I used it for, and what results I got. If I think a tool isn't worth the money, I say so — even if it has an affiliate program.
